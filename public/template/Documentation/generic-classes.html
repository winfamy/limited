Generic Class Reference

Following helper classes can be used with any elements in order to help remove repetition or override certain classes/IDs. Using helper classes will make your code more reusable and scalable for many features that will be added in the future.

e.g:

<div class="m-t-20 text-center">
    ...
</div>

<h2 class="title text-uppercase p-10">
    ...
</h2>
