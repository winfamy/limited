Calendar

FullCalendar is a JavaScript event calendar. It's fully customizable and open source.

Official website and detailed documentation at: https://fullcalendar.io/

Live preview: /calendar.html


In order to use FullCalendar in your web page, you'll have to include it's CSS and Javascript asset files. FullCalendar requires MomentJs for it's date/time functionality. Make sure to include MomentJs before FullCalendar JS library.

<!-- CSS -->
<link href="vendors/bower_components/fullcalendar/dist/fullcalendar.min.css" rel="stylesheet">

<!-- JS -->
<script src="vendors/bower_components/moment/min/moment.min.js"></script>
<script src="vendors/bower_components/fullcalendar/dist/fullcalendar.min.js"></script>

Basic example

  • HTML
  • Javascript
<div id="calendar"></div>
$('#calendar').fullCalendar({
    // put your options and callbacks here
})


Source files:

  • Plugin:
    • CSS: /vendors/bower_components/fullcalendar/dist/fullcalendar.min.css
    • JS:
      /vendors/bower_components/moment/min/moment.min.js
      /vendors/bower_components/fullcalendar/dist/fullcalendar.min.js
  • LESS: /less/inc/vendor-overrides/fullcalendar.less
  • HTML: /calendar.html
