Wizard

This twitter bootstrap plugin builds a wizard out of a formatter tabbable structure. It allows to build a wizard functionality using buttons to go through the different wizard steps and using events allows to hook into each step individually. This utilizes Bootstrap's Tab component to build it's UI structure.

Official website and detailed documentation at: https://github.com/VinceG/twitter-bootstrap-wizard

Live preview: /components.html

In order to use this component, make sure to include it's javascript asset file.

<script src="vendors/bower_components/twitter-bootstrap-wizard/jquery.bootstrap.wizard.min.js"></script>

Basic example

<div class="tab-wizard">
    <ul class="tab-nav tab-nav--centered tab-wizard__nav">
        <li class="active"><a href="#tab-wizard-1" data-toggle="tab">First</a></li>
        <li><a href="#tab-wizard-2" data-toggle="tab">Second</a></li>
        <li><a href="#tab-wizard-3" data-toggle="tab">Third</a></li>
        <li><a href="#tab-wizard-4" data-toggle="tab">Forth</a></li>
        <li><a href="#tab-wizard-5" data-toggle="tab">Fifth</a></li>
        <li><a href="#tab-wizard-6" data-toggle="tab">Sixth</a></li>
    </ul>

    <div class="tab-content">
        <div class="tab-pane active fade in" id="tab-wizard-1">...</div>
        <div class="tab-pane fade" id="tab-wizard-2">...</div>
        <div class="tab-pane fade" id="tab-wizard-3">...</div>
        <div class="tab-pane fade" id="tab-wizard-4">...</div>
        <div class="tab-pane fade" id="tab-wizard-5">...</div>
        <div class="tab-pane fade" id="tab-wizard-6">...</div>

        <ul class="pagination pagination--alt">
            <li class="tab-wizard__first tab-wizard__previous"><span><i class="zmdi zmdi-more-horiz"></i></span></li>
            <li class="tab-wizard__previous"><span><i class="zmdi zmdi-long-arrow-left"></i></span></li>
            <li class="tab-wizard__next"><span><i class="zmdi zmdi-long-arrow-right"></i></span></li>
            <li class="tab-wizard__last tab-wizard__next"><span><i class="zmdi zmdi-more-horiz"></i></span></li>
        </ul>
    </div>
</div>
