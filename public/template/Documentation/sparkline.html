jQuery Sparkline

This jQuery plugin generates sparklines (small inline charts) directly in the browser using data supplied either inline in the HTML, or via javascript.

Official website and detailed documentation at: http://omnipotent.net/jquery.sparkline/

Live preview: /other-charts.html


In order to use this component, make sure to include it's javascript asset file.

<script src="vendors/jquery.sparkline/jquery.sparkline.min.js"></script>

Sparkline Bar Chart

  • HTML
  • Javascript
<div class="chart-sparkline-bar">6,9,5,6,3,7,5,4,6,5,6,4,2,5,8,2,6,9,5,7,2,5,2,8,6,7,6,5,3,1,9,3,5,8,2,4</div>
$('.chart-sparkline-bar').sparkline('html', {
    type: 'bar',
    height: 40,
    barWidth: 3,
    barColor: '#b4bfc3',
    barSpacing: 2
})

Sparkline Line Chart

  • HTML
  • Javascript
<div class="chart-sparkline-line">9,5,6,3,9,7,5,4,6,5,6,4,9</div>

$('.chart-sparkline-line').sparkline('html', {
    type: 'line',
    width: '100%',
    height: 50,
    lineColor: rgba(255, 255, 255, 0.2)',
    fillColor: 'rgba(0,0,0,0)',
    lineWidth: 1.5,
    maxSpotColor: '#b4bfc3',
    minSpotColor: '#b4bfc3',
    spotColor: '#b4bfc3',
    spotRadius: 4,
    highlightSpotColor: '#b4bfc3',
    highlightLineColor: '#b4bfc3'
});
