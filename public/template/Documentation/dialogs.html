SweetAlert2 Dialogs

A beautiful and customizable replacement for JavaScript's popup boxes.

Official website and detailed documentation at https://limonte.github.io/sweetalert2/

In order to check the previews of these notifications, please visit the page /notification-dialog.html


In order to use these plugin, make sure to include it's CSS and Javascript asset files in the <head> section.

<!-- CSS -->
<link href="vendors/bower_components/sweetalert2/dist/sweetalert2.min.css" rel="stylesheet">

<!-- JS -->
<script src="vendors/bower_components/sweetalert2/dist/sweetalert2.min.js"></script>

Basic Example

  • HTML
  • Javascript
<button class="btn btn--light" id="sa-basic">Click me</button>
$('#sa-basic').click(function(){
    swal("Here's a message!");
});

A warning message with a function

  • HTML
  • Javascript
<button class="btn btn--light" id="sa-function">Click me</button>
$('#sa-function').click(function(){
    swal({
        title: "Are you sure?",
        text: "You will not be able to recover this imaginary file!",
        type: "warning",
        showCancelButton: true,
        confirmButtonColor: "#DD6B55",
        confirmButtonText: "Yes, delete it!"
    }).then(function(){
        swal(
            "Deleted!",
            "Your imaginary file has been deleted.",
            "success"
        );
    });
});
