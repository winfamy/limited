Icons

Material Design Iconic Font is a full suite of material design icons (created and maintained by Google) with additional community-designed and brands icons for easy scalable vector graphics on websites or desktop.

Official website and detailed documentation: https://zavoloklom.github.io/material-design-iconic-font/index.html

You can check all the available font icons at /icons.html


In order to use these icons, first you'll have to include it's CSS asset file inside the <head> tag.

<link href="vendors/bower_components/material-design-iconic-font/dist/css/material-design-iconic-font.min.css" rel="stylesheet">

Basic usage esample

<!-- zmdi zmdi-[icon-name] -->

<i class="zmdi zmdi-home"></i>
