Containers

Material Admin requires a containing element to wrap site contents in order to make it center aligned. There are 2 different variant of containers available in Material Admin. Note that, due to padding and more, neither container is nestable.

  • No Container(default)
  • Container boxed
  • Container boxed small

<div id="content">
    <!-- contents -->
</div>

Usage examples:

  • /index.html
  • /typography.html
  • /widgets.html
  • /tables.html
  • etc.
<div id="content" class="content--boxed">
    <!-- contents -->
</div>

Usage examples:

  • /messages.html
  • /calendar.html
  • /pricing-table.html
<div id="content" class="content--boxed-sm">
    <!-- contents -->
</div>

Usage examples:

  • /profile-timeline.html
  • /profile-about.html
  • /pricing-photos.html
  • /wall.html

Page Headers

Page header is a wrapper for page titles and optional page actions.

  • Basic page header
  • Page header with text
  • Page header with actions
<section id="content">
    <div class="content__header">
        <h2>This is a page header</h2>
    </div>
</section>
<div id="content">
    <div class="content__header">
        <h2>This is a page header <small>this is a sample page header sub text</small></h2>
    </div>
</div>
<div id="content">
    <div class="content__header">
        <h2>This is a page header <small>this is a sample page header sub text</small></h2>

        <div class="actions">
            <a href=""><i class="zmdi zmdi-search"></i></a>
            <a href=""><i class="zmdi zmdi-home"></i></a>
        </div>
    </div>
</div>

You can also use the helper classes in order to align the title to right or center.

<div id="content">
    <div class="content__header text-right"><!-- or .text-center -->
        <h2>This is a page header</h2>
    </div>
</div>

Action Icons

Action Icons are used to perform a certain list of actions within a title, list or in any element using icons.

<div class="actions">
    <a href=""><i class="zmdi zmdi-favorite-outline"></i></a>
    <a href=""><i class="zmdi zmdi-print"></i></a>
    <a href=""><i class="zmdi zmdi-email"></i></a>
    <span class="dropdown">
        <a href="" data-toggle="dropdown"><i class="zmdi zmdi-share"></i></a>

        <ul class="dropdown-menu">
            <li></li>
        </ul>
    </span>
</div>

To override position of the action-icons, you can simply add a modifier class on .actions and customize it as per your need.

e.g HTML

<div class="actions actions--list">
   ...
</div>

e.g CSS

.actions--list {
    float: right;
    margin-top: 10px;
    ...
}

List Groups

List Group modifier classes

  1. .list-group--striped: Adds zebra-striping to all even list group items within list group.

    <div class="list-group list-group--striped">
    ...
</div>

  2. .list-group--light: Transforms default list-group in to a light colored theme.

    <div class="list-group list-group--light">
    ...
</div>

Other List Group helper classes

  1. .list-group__checkbox: Aligns main checkboxes inside list-group-item
  2. .list-group__empty: Adds empty space end of list-group
