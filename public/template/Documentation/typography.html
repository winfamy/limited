Typography

Body Copy

Material Admin's global default font-size is 14px, with a line-height of 1.42857143. This is applied to the body and all paragraphs. In addition, p (paragraphs) receive a bottom margin of 20px.

<p>...</p>

<p class="lead">...</p>

Heading

All HTML headings, h1 through h6, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>
<h4>This is a Heading 4</h4>
<h5>This is a Heading 5</h5>
<h6>This is a Heading 6</h6>

Inline Text Elements

An inline element does not start on a new line and only takes up as much width as necessary.

<mark>This is marked text</mark>
<del>This is Deleted Text</del>
<ins>This is Inserted Text</ins>
<small>This is Small Text</small>
<strong>This is Bold Text</strong>
<s>This is Strikethrough Text</s>
<u>This is Underlined Text</u>
<em>This is Italic Text</em>

Abbreviations

Stylized implementation of HTML's 'abbr' element for abbreviations and acronyms to show the expanded version on hover. Abbreviations with a 'title' attribute have a light dotted bottom border and a help cursor on hover, providing additional context on hover and to users of assistive technologies.

<abbr title="attribute">attr</abbr>
<abbr title="HyperText Markup Language" class="initialism">HTML</abbr>

Blockquotes

For quoting blocks of content from another source within your document.

<blockquote>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
</blockquote>

<blockquote>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
  <footer>Someone famous in <cite title="Source Title">Source Title</cite></footer>
</blockquote>

Unordered Lists

A list of items in which the order does not explicitly matter.

<ul class="un-list un-list--angle">
   ...
</ul>

<ul class="un-list un-list--check">
   ...
</ul>

<ul class="un-list un-list--star">
   ...
</ul>
